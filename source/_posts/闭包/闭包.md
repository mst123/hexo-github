看着简单，实际隐藏了很多东西
```
function count(){
    let countNum = 0
    return () => {
        return ++countNum
    }
}
console.log((count())()); //1
console.log((count())()); //1
console.log((count())()); //1
let countFunc = count()
console.log(countFunc()); //1
console.log(countFunc()); //2
console.log(countFunc()); //3
```
直接写成自执行函数，更加简单
```
const countUp = (() => {
  let count = 0;
  return () => {
    return ++count;
  };
})();
console.log(countUp()); // 1
console.log(countUp()); // 2
```
